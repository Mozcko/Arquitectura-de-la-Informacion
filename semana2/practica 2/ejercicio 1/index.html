Conceptos básicos de HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Anatomía de un elemento HTML

    Las partes principales del elemento son:

  1. La etiqueta de apertura: consiste en el nombre del elemento, encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento.
  2. etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento.
  3. El contenido: es la parte de contenido que te interesa que vea el usuario de una forma particular, como: párrafo, imagen, lista, tabla, etc.
  4. El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.
    • Los elementos pueden también tener atributos.
    • Los atributos contienen información adicional acerca del elemento, la cual no quieres que aparezca en el contenido real del elemento.

    atributo debe tener siempre:

  • espacio entre el atributo y el nombre del elemento (o del atributo previo, si el elemento ya posee uno o más atributos).
  • nombre del atributo, seguido por un signo de igual (=).
  • de apertura y de cierre, encerrando el valor del atributo.
  • Los atributos siempre se incluyen en la etiqueta de apertura de un elemento, nunca en la de cierre.
